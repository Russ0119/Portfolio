Capstone Project

Physicochemical Water Parameter Monitoring and Recommendation System

This capstone project centers on the development of a web-based Physicochemical Water Parameter Monitoring and Recommendation System for aquaponics. The system monitors various water parameters and provides recommendations on whether a water change is needed, with suggestions tailored to the specific conditions of each parameter.

Key Features

  • Provides Live Data Such As the Water Parameters' Value, Graph in the Dashboard, General Assessment, and Recommendation.
  • Generate Water Report, and Export Report Feature.
  • Provides Live Notifications.
  • Provides User's Activity Logs.
  • Provide users the Baseline/Optimal Range for Each Parameters.
Software Used: Visual Studio (With Codeigniter 4 Framework), Microsoft Azure, Arduino IDE
Hardware Used: Laptop, Arduino Uno, Wemod R1 D1, Water Sensors.
